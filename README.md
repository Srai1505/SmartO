# 🚀 SmartO - Smart To-Do App

SmartO is a JavaScript-based to-do application that helps you manage daily tasks while understanding real-world web app logic. 💡

## ✨ Features

* ➕ Add tasks easily
* ❌ Delete tasks anytime
* ✅ Mark tasks as complete
* 🌙 Dark mode for better UI
* 🔥 Priority setting for tasks

## 🧠 Why SmartO?

SmartO is not just a task manager—it’s a learning tool! It demonstrates how modern web applications work, including dynamic updates and user interaction.

## 🛠️ Built With

* HTML
* CSS
* JavaScript

## 📌 Usage

Simply add your tasks, organize them by priority, and track your progress efficiently.



💻 Stay productive & keep learning with SmartO!
